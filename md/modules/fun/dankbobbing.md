## DankBobbing

DankBobbing exaggerates Minecraft's normal view bobbing — the slight up-and-down sway of your hand and screen as you walk. With it enabled, that motion is amplified into a wild, over-the-top wobble. It's purely a visual gag with no gameplay benefit, made for fun, recording clips, or just messing around.

The **Motion** setting controls how intense the effect is: higher values produce a more dramatic, swingy bob. Note that this only affects what you see locally — it doesn't change how you actually move or how other players see you. If you instead want to remove view bobbing entirely, see [NoBob](/docs/modules/render/nobob).

**Category:** Fun
**Enabled by default:** No

### Settings

| Setting | Type | Default | Range | Description |
|---------|------|---------|-------|-------------|
| Motion | Decimal | 5.0 | 1.0..50.0 | How strongly the view bobbing is amplified. Higher values make the bobbing more extreme. |
