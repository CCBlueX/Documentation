## FastBreak

Allows you to break blocks faster.

**Category:** World  
**Enabled by default:** No  

### Settings

Below is the complete tree of all configurable settings for this module.

```
├── BreakDamage (Decimal | default: 0.8 | range: 0.1..1.0)
├── OnlyTool (Toggle | default: false)
└── Mode (Mode Selector | default: None | modes: None, AbortAnother)
```

### Settings Details

- **BreakDamage** (Decimal) — default: `0.8`; range: `0.1` – `1.0`
- **OnlyTool** (Toggle) — default: `false`
#### Mode

Select a mode for this feature. Available modes: **None**, **AbortAnother**. Default: **None**.
