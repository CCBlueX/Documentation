## EntityControl

Allows you to control rideable entities without a saddle.

**Category:** Movement  
**Enabled by default:** No  

### Settings

Below is the complete tree of all configurable settings for this module.

```
└── Enforce (Multi-Select | default: [Saddled, JumpStrength] | options: Saddled, JumpStrength)
```

### Settings Details

- **Enforce** (Multi-Select) — default: `Saddled`, `JumpStrength`; options: `Saddled`, `JumpStrength` — Selects which checks to bypass: Saddled makes the entity act as if saddled, JumpStrength forces maximum jump strength.
