## AntiBounce

AntiBounce stops you from bouncing on bouncy blocks. Normally, landing on a slime block (or other springy surface) launches you back into the air, which can fling you off course, break your timing, or send you somewhere you didn't intend. With this module on, you simply land and stay put instead of springing back up.

This is useful when you're moving across terrain that includes slime blocks and you want predictable, grounded movement — for example while running parkour-style routes, fleeing across a base, or fighting near bounce pads where an unexpected launch would leave you vulnerable. It is off by default, so enable it only when you actually need to suppress bouncing.

AntiBounce has no options to configure — turning it on is all that's required.

**Category:** Movement
**Enabled by default:** No

### Settings

This module has no configurable settings.
