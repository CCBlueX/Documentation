## ProjectileAimbot

Automatically aims your held projectile weapon at the best nearby target. It calculates the correct launch angle for whatever projectile you are holding (bow, crossbow, trident, snowball, egg, etc.) and rotates toward the target so your shot connects.

**Category:** Combat
**Enabled by default:** No

### Settings

Below is the complete tree of all configurable settings for this module.

```
├── Target (Setting Group → see Shared: Target)
└── Rotations (Setting Group → see Shared: Rotations)
```

### Settings Details

#### Target

> Selects which entity to aim at. These are the shared target settings. See [Shared: Target](/docs/modules/shared/target).

#### Rotations

> Controls how the server-side aim moves toward the target. These are the shared rotation settings. See [Shared: Rotations](/docs/modules/shared/rotations).
